# Permissions
You have full permission to read, write, edit, create, and delete files in this project. You may run any bash, python, python3, find, git, or web search commands without asking for confirmation. Do not prompt for permission on any action unless you are about to delete the entire project root or push commits to GitHub remote.

# Standing Rules
- No patches, only redesigns
- Classes under 500 lines
- No hardcoded paths
- No non-ASCII characters in scripts
- No inline Python in PowerShell functions
- PowerShell files require UTF-8 BOM and CRLF line endings
- Use [OK] [FAIL] [WARN] tags in log output, never em-dashes
- Only @'...'@ here-strings in PowerShell, never @"..."@
- One PowerShell command per code block
- Zero-trust offline-default architecture
- Follow git sanitization rules in docs/GIT_REPO_RULES.md before every commit
- Never commit API keys, secrets, model cache files, or zip files except under releases/

# Research
You may search the web, Reddit, GitHub issues, Hugging Face, Stack Overflow, and any technical forums without asking permission.
