B. HUGGINGFACE RETIREMENT
=============================================================

MOTIVATION:
    - HuggingFace sentence-transformers requires AI Use Case approval
    - Ollama already serves our LLM, can also serve embeddings
    - nomic-embed-text (768-dim) is higher quality than all-MiniLM-L6-v2 (384-dim)
    - Removes ~2.5GB of dependencies (torch, transformers, etc.)
    - Apache 2.0 license, no approval needed

FILE: src/core/embedder.py (FULL REWRITE)
    WHAT: Replaced SentenceTransformer class with Ollama HTTP client.
          Same public API preserved: embed_batch(), embed_query(),
          encode(), close(), dimension property.
    HOW:  Sends text to Ollama /api/embed endpoint via httpx.Client.
          Batches configurable via HYBRIDRAG_EMBED_BATCH env var (default 64).
          L2-normalizes output so dot product = cosine similarity.
          Startup probe detects dimension and verifies Ollama is running.

FILE: requirements.txt, requirements_approved.txt
    WHAT: Removed 16 HuggingFace packages:
          sentence-transformers, torch, transformers, tokenizers,
          huggingface_hub, safetensors, scipy, scikit-learn, sympy,
          threadpoolctl, einops, joblib, filelock, fsspec, networkx,
          mpmath, Jinja2, MarkupSafe
    WHY:  No longer needed. Install size dropped from ~800MB to ~200MB.

FILE: config/default_config.yaml
    WHAT: embedding.model_name: nomic-embed-text (was all-MiniLM-L6-v2)
          embedding.dimension: 768 (was 384)

FILE: src/core/config.py (EmbeddingConfig dataclass)

FILE: src/core/hallucination_guard/guard_types.py
